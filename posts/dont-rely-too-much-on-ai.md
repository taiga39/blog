I don't want to rely too much on AI.
I joined a new project, so I started using Go for the first time.
I use a coding agent, but I feel I rely on it too much.
In my case, it is better to write code by myself than to let AI write it.
Because it is not good for learning, and the design is not very good yet.
I still need to review everything.
Also, I don't know Go well.
But I think I understand better where to write things.
Even if it is my first time using clean architecture.
So I think I will use AI more after I can give better instructions.
