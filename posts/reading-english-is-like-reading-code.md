Reading English is similar to reading programming code.
I think reading English and reading programming code are similar.

Recently, I have been using Go. At first, I didn’t understand everything, like conventions and syntax, when I read code.

However, as I continued to use Go while writing code and doing code reviews, I started to understand it better.

Now I am studying English, and the same thing is happening.

Both are languages, and it is important to get used to them.
