I removed the smart device from my wrist.

I have used a Fitbit for six years.

It was useful for notifications, alarms, timers, and counting steps. But almost all of those features are not really necessary.

Sleep logs are similar to how I feel, so it is not necessary to measure them.

However, a digital timer on my wrist was useful for cooking and keeping track of waiting time at cafés. But because I could change the alarm time quickly, I often went back to sleep.

I've also been working on a desk-cleaning project, so I want to reduce the number of charging cables.
